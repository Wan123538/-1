# 🎲 随机抽奖/抽签工具

一个简洁、美观且功能完整的随机抽奖工具，适用于活动抽奖、随机分组、决策选择等场景。

## ✨ 功能特点

- 📝 **输入功能**：支持多行文本输入，每行一个候选者
- 🎯 **数量设置**：自定义抽取数量
- 🔀 **公平算法**：使用 Fisher-Yates 洗牌算法保证公平
- 🎊 **结果展示**：动画展示中奖名单
- 🔄 **重置功能**：一键重置重新开始
- 📱 **响应式设计**：完美适配移动端和桌面端

## 📦 安装运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 📖 使用说明

1. 在输入框中输入候选者名单，每行一个
2. 设置要抽取的数量
3. 点击「开始抽奖」按钮
4. 等待抽奖动画完成，查看中奖结果
5. 点击「重置」可重新开始
