# 幸运抽奖 - 随机抽签工具

一个精美且功能完整的在线随机抽奖/抽签工具，帮助您快速从多个选项中随机抽取中奖者。

## ✨ 功能特点

- 📝 **灵活输入**：支持自定义选项输入，可批量粘贴
- 🎯 **预设模板**：内置年会抽奖、课堂点名、团队活动等模板
- ⚙️ **灵活配置**：可设置抽取人数，支持重复/不重复抽取模式
- 🎰 **精美动画**：流畅的抽奖动画效果，提升抽奖体验
- 📜 **历史记录**：自动保存抽奖历史，随时回顾
- 📋 **一键复制**：快速复制抽奖结果
- 📱 **响应式设计**：完美适配桌面端和移动端

## 🚀 快速开始

### 本地运行

1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖
3. 无需配置服务器

### GitHub Pages 部署

1. 将此仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支
4. 等待部署完成（约1-2分钟）
5. 访问 `https://[username].github.io/[repository-name]/`

## 🎮 使用方法

1. **输入选项**：在文本框中输入抽奖选项，每行一个
2. **选择模板**（可选）：点击预设模板快速填充
3. **设置人数**：调整抽取人数（默认1人）
4. **选择模式**：开关重复抽取功能
5. **开始抽奖**：点击金色抽奖按钮
6. **查看结果**：中奖名单将以卡片形式展示
7. **操作结果**：可重新抽取、复制结果或清空重置

## 🛠️ 技术栈

- React 18 (CDN)
- CSS3 动画
- Fisher-Yates 随机算法
